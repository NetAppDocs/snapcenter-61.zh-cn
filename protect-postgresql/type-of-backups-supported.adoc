= 支持的备份类型
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
备份类型指定您要创建的备份类型。  SnapCenter支持 PostgreSQL 集群的基于快照复制的备份类型。



== 基于快照副本的备份

基于快照副本的备份利用NetApp快照技术来创建 PostgreSQL 集群所在卷的在线只读副本。
