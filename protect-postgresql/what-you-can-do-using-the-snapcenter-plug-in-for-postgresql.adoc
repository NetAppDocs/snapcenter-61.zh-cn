= 使用适用于 PostgreSQL 的SnapCenter插件可以做什么
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在您的环境中安装适用于 PostgreSQL 集群的插件时，您可以使用SnapCenter备份、还原和克隆 PostgreSQL 集群及其资源。您还可以执行支持这些操作的任务。

* 添加集群。
* 创建备份。
* 从备份中恢复。
* 克隆备份。
* 安排备份操作。
* 监控备份、恢复和克隆操作。
* 查看备份、恢复和克隆操作的报告。
