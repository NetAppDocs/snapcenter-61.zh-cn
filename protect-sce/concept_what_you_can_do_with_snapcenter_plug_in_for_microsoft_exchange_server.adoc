= SnapCenter插件 for Microsoft Exchange Server 的功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 Exchange 插件来备份和恢复 Exchange Server 数据库。

* 查看和管理 Exchange 数据库可用性组 (DAG)、数据库和副本集的活动清单
* 定义提供备份自动化保护设置的策略
* 为资源组分配策略
* 保护单个 DAG 和数据库
* 备份主 Exchange 邮箱数据库和辅助 Exchange 邮箱数据库
* 从主备份和辅助备份恢复数据库
