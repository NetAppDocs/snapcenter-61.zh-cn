= 如何本机访问SnapCenter REST API
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用任何支持 REST 客户端的编程语言直接访问SnapCenter REST API。流行的语言选择包括 Python、PowerShell 和 Java。
