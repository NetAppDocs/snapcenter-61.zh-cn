= 使用 CLI 卸载 Linux 或 AIX 的插件包
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用命令行界面卸载适用于 Linux 的SnapCenter插件包或适用于 AIX 的SnapCenter插件包。

.开始之前
* 确保已删除计划作业
* 确保所有正在运行的作业都已完成。


.步骤
运行 _/custom_location/ NetApp/snapcenter/spl/installation/plugins/uninstall_ 进行卸载。
